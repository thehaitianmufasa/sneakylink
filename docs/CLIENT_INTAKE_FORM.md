### Purpose
This document provides all questions needed to create a Google Form for collecting client information. The form responses will be used to generate the client's `config.json` file automatically.

### Form Setup Instructions
1. Go to: https://docs.google.com/forms
2. Click "Blank Form" or use template
3. Title: "NeverMissLead Website Setup Form"
4. Description: "Help us build your perfect lead generation website! Please answer all questions so we can create a custom site for your business."
5. Add all questions below in order
6. Set up Google Sheets integration for automatic responses

### Response Time
- **Estimated time for client to complete**: 15-20 minutes
- **Submission**: Client submits ‚Üí You receive email notification ‚Üí Download responses as CSV

## üîÑ CONVERTING FORM RESPONSES TO CONFIG.JSON

### Option A: Manual (5 minutes)
1. Download Google Sheets as CSV
2. Open `/config/config.template.json`
3. Fill in fields using CSV data
4. Save as `/config/clients/[slug].json`

### Option B: Automated (1 minute)
Use the `generate-config.js` script:

```bash
# Download Google Sheets as CSV
# Run automation script
node scripts/generate-config.js \
  --input=responses.csv \
  --row=2 \  # Which row to process (2 = first response)
  --output=config/clients/client-slug.json

# Validate output
npm run validate-config config/clients/client-slug.json
```
